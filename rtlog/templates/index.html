RTChatLog : Realtime IRC Log

Toy project to learn web push technology. Currently only use long-polling techique.
I create simple IRC bot that join to some public IRC channels. This bot will listen to all conversation, and send the message to rtchatlog server.
You can use #rtchatlog at irc.freenode.net as a test channel, and see the output in this page

Realtime Log using Long-polling

Use long polling technique from https://bitbucket.org/danjac/chatbox/.
It use flask and gevent
